Website Integration

Integrate your LLM-powered chat agent into any website with a simple script tag. Follow these steps to get started:

1. Create an Agent

Go to the Create Agent tab and configure your agent with a name, description, and optional logo.

2. Add the Script to Your Website

Copy the generated script tag and paste it into the <head> section of your HTML:

<script src="https://llm-data-platform.example.com/agent.js?id=YOUR_AGENT_ID" async></script>
3. Customize Behavior (Optional)

You can customize the chat widget by adding a configuration object before the script:

<script>
  window.LLMChatConfig = {
    position: 'bottom-right',  // bottom-right, bottom-left, top-right, top-left
    initialMessage: 'Hello! How can I help you today?',
    primaryColor: '#0d6efd',
    textColor: '#ffffff',
    showBranding: false
  };
</script>

Telegram Bot Integration

Create a Telegram bot that uses your LLM Data Platform with rug pull capabilities. This allows you to reach users through the Telegram messaging platform.

1. Generate an API Key

Go to the Telegram Bot tab and click "Generate API Key".

2. Create a Bot with BotFather

Open Telegram and search for @BotFather. Start a chat and use the /newbot command to create a new bot. BotFather will give you a token.

3. Set Up Your Bot

Create a new Node.js project and install the required dependencies:

npm init -y
npm install telegraf axios dotenv
4. Create Environment File

Create a .env file with your Telegram bot token:

TELEGRAM_BOT_TOKEN=your_bot_token_from_botfather
5. Use the Generated Bot Code

Copy the generated bot code from the Telegram Bot tab and save it as bot.js.

6. Run Your Bot

Start your bot with:

node bot.js

Python Integration

Use your LLM Data Platform with Python applications by leveraging our API:

1. Install Required Packages
pip install requests
2. Basic API Usage
import requests

# Replace with your API key and endpoint
API_KEY = "your_api_key"
API_URL = "https://llm-data-platform.example.com/api"

# Send a query to the LLM
def ask_question(question):
    response = requests.post(
        f"{API_URL}/chat",
        json={
            "message": question,
            "apiKey": API_KEY
        }
    )
    
    if response.status_code == 200:
        return response.json()["answer"]
    else:
        return f"Error: {response.status_code} - {response.text}"

# Example usage
question = "What is the capital of France?"
answer = ask_question(question)
print(f"Q: {question}\nA: {answer}")
3. Telegram Bot with Python

You can also create a Telegram bot using Python:

import requests
import os
from telegram import Update
from telegram.ext import ApplicationBuilder, CommandHandler, MessageHandler, filters, ContextTypes

# Replace with your API key and endpoint
API_KEY = "your_api_key"
API_URL = "https://llm-data-platform.example.com/api"

# Get your Telegram bot token from environment variable
TELEGRAM_TOKEN = os.environ.get("TELEGRAM_TOKEN")

async def start(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text('Hello! I am your AI assistant powered by LLM Data Platform. Ask me anything!')

async def help_command(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text('Just send me a message and I will respond using advanced AI.')

async def handle_message(update: Update, context: ContextTypes.DEFAULT_TYPE):
    message = update.message.text
    user_id = update.effective_user.id
    
    # Show typing indicator
    await context.bot.send_chat_action(chat_id=update.effective_chat.id, action="typing")
    
    # Call the LLM Data Platform API
    try:
        response = requests.post(
            f"{API_URL}/telegram/chat",
            json={
                "message": message,
                "telegramUserId": str(user_id),
                "apiKey": API_KEY,
                "chatContext": {
                    "chatId": str(update.effective_chat.id),
                    "username": update.effective_user.username or "",
                    "firstName": update.effective_user.first_name or "",
                    "lastName": update.effective_user.last_name or ""
                }
            }
        )
        
        if response.status_code == 200:
            answer = response.json().get("answer", "Sorry, I could not generate a response.")
        else:
            answer = f"Sorry, I encountered an error. Please try again later."
    except Exception as e:
        answer = f"Sorry, I encountered an error. Please try again later."
    
    await update.message.reply_text(answer)

def main():
    application = ApplicationBuilder().token(TELEGRAM_TOKEN).build()
    
    application.add_handler(CommandHandler("start", start))
    application.add_handler(CommandHandler("help", help_command))
    application.add_handler(MessageHandler(filters.TEXT & ~filters.COMMAND, handle_message))
    
    application.run_polling()

if __name__ == '__main__':
    main()
